# Shelf Solver — 最大货架可见面优化

## 问题描述

### 非形式化描述

在一个网格化的商店地图中，存在一个入口 $S$ 和一个出口 $T$。地图上的每个空闲格子可以放置一个货架，也可以作为通道供顾客行走。顾客从 $S$ 出发，沿通道行走到达 $T$。货架与顾客路径相邻的面称为「可见面」（face）。目标是：在保证 $S$ 到 $T$ 连通的前提下，选择一种货架摆放方案，使得顾客路径上能看到的总可见面数最大。

### 形式化定义

给定一个 $W \times H$ 的网格图 $G = (V, E)$，其中：

- $V_{\text{free}} \subseteq V$ 为所有可用格子的集合（不含墙壁/障碍物）。
- $S, T \in V_{\text{free}}$ 分别为起点和终点，且 $S \neq T$。
- 邻接关系 $\mathcal{N}(v)$ 根据所选邻域模型（N8/N4/N2/N1）定义每个格子 $v$ 的邻居集合。

**决策变量：** 对每个 $v \in V_{\text{free}} \setminus \{S, T\}$，定义

$$
x_v \in \{0, 1\}, \quad x_v =
\begin{cases}
1, & \text{若在 } v \text{ 处放置货架} \\
0, & \text{若 } v \text{ 为通道}
\end{cases}
$$

**通道集合** $P = \{S, T\} \cup \{v \in V_{\text{free}} \mid x_v = 0\}$。

**可行性约束：** 在通道子图 $G[P]$ 中，$S$ 与 $T$ 必须连通。即存在一条路径 $\pi = (v_0 = S, v_1, \dots, v_k = T)$，其中 $v_i \in P$ 且 $v_i$ 与 $v_{i+1}$ 四连通相邻。

**覆盖度（Coverage）目标函数：** 设 $\pi$ 为 $G[P]$ 中 $S \to T$ 的最短路径（BFS），定义路径上的格子集合 $\Pi = \{v_0, \dots, v_k\}$，以及每个货架格子的可见面计数：

$$
f(v) = \left| \mathcal{N}(v) \cap \Pi \right|, \quad \forall v \text{ s.t. } x_v = 1
$$

**优化目标：**

$$
\max_{\mathbf{x}} \sum_{v : x_v = 1} f(v) \quad \text{s.t.} \quad S \xrightarrow{G[P]} T
$$

### 邻域模型变体

| 模型 | $\mathcal{N}(v)$ | 含义 |
|------|---|---|
| **N8** (Moore) | 周围 8 个格子 | 货架所有面均可被看到 |
| **N4** (Von Neumann) | 上下左右 4 个格子 | 仅正交方向可见 |
| **N2** | 两个对向面 | 前后两面，取最优朝向 |
| **N1** | 单一面 | 仅正面可见，取最优朝向 |

### 2-cell 变体（Domino 模式）

在 2-cell 模式下，货架占据 $1 \times 2$ 的矩形（多米诺骨牌）。贪心配对后，未成功配对的单格货架降级为**路障**（roadblock）：它们仍然阻塞通道（不可通行），但不贡献任何可见面得分。覆盖度计算基于已配对多米诺骨牌的外周面。

## Write-Up：算法分析与求解策略

### 1. 问题本质

本问题是一个**带连通性约束的组合优化问题**。将网格中的空闲格子分为两个角色——货架与通道——使得通道保持 $S$–$T$ 连通，且货架对通道的"曝光量"最大化。

该问题可以等价地转化为：**寻找一条从 $S$ 到 $T$ 的路径 $\pi$，使得路径以外的空闲格子（作为货架）对该路径的总可见面数最大。** 这是因为给定一条可行路径后，最优策略是将所有非路径的空闲格子全部设为货架——额外的货架不会减少任何已有货架的得分，只会增加（或持平）总可见面数。

> **注：** 在 N1/N2 等非对称邻域模型下，某些边缘情况中"空出"一些格子理论上可能改变 BFS 最短路径的走向，从而间接影响得分。但本算法通过在 SA 中直接搜索路径空间来隐式处理这种耦合，因此上述简化在实践中有效。

### 2. 搜索空间转换

基于上述观察，算法将搜索空间从 $\{0,1\}^{|V_{\text{free}}|}$（指数级的 0/1 向量空间）压缩为**所有 $S$–$T$ 路径的集合**。给定任意一条 $S$–$T$ 路径 $\pi$：

1. 令 $\Pi = \{v \in \pi\}$ 为路径上的格子集合。
2. 货架配置 $x_v = [\,v \notin \Pi\,]$（路径外的格子全部放货架）。
3. 通过 BFS 重新计算通道中的最短路径，得到最终实现路径。
4. 计算覆盖度 $\sum_{v:x_v=1} f(v)$。

### 3. 模拟退火（Simulated Annealing）

算法采用**模拟退火**作为核心元启发式搜索策略：

**初始解：**
- 第 0 次重启使用 BFS 最短路径作为基线——它提供路径长度最优的起点。
- 后续重启使用**随机 DFS** 生成拓扑结构多样化的初始路径，从而探索不同的解空间区域。

**邻域操作（Reroute）：**
1. 在当前路径 $\pi = (v_0, v_1, \dots, v_k)$ 上随机选取两个位置 $0 \le i < j \le k$。
2. 在由以下格子构成的子图上，使用随机 DFS 搜索从 $v_i$ 到 $v_j$ 的替代路径：
   - 旧路径段 $\{v_{i+1}, \dots, v_{j-1}\}$（允许复用）
   - 当前不在路径上的空闲格子（允许借道）
3. 用新子路径替换旧段，生成候选路径 $\pi'$。

**接受准则（Metropolis）：**

$$
P(\text{accept}) =
\begin{cases}
1, & \text{if } \Delta \ge 0 \\
e^{\Delta / T}, & \text{if } \Delta < 0
\end{cases}
$$

其中 $\Delta = \text{coverage}(\pi') - \text{coverage}(\pi)$，温度 $T$ 按几何退火降温：

$$
T_{k+1} = \alpha \cdot T_k, \quad \alpha = \left(\frac{T_{\min}}{T_0}\right)^{1/N}
$$

$N$ 为总迭代次数，$T_0$ 和 $T_{\min}$ 分别为初始温度和终止温度。

### 4. 多重启与并行化

为避免陷入局部最优，算法执行多次独立重启（默认 8 次），每次使用不同的随机种子和初始路径。所有重启可在多线程中并行执行，最终取覆盖度最高的解。

### 5. 评估流程

对于每条候选路径 $\pi$：

1. **构建货架布局：** 路径格子为通道，其余为货架。
2. **BFS 重路由：** 在通道子图上计算 $S \to T$ 的最短路径 $\pi^*$。
3. **计分：** 遍历每个货架格子，统计其邻域与 $\pi^*$ 的交集大小并求和。

在 2-cell 模式下，额外执行贪心配对：将相邻货架格子匹配为多米诺骨牌，未配对者降为路障。

### 6. 复杂度分析

| 操作 | 复杂度 |
|------|------|
| BFS 最短路径 | $O(WH)$ |
| 随机 DFS（reroute） | $O(WH)$（最坏） |
| 单次覆盖度评估 | $O(\|V_{\text{free}}\| \cdot \|\mathcal{N}\|)$ |
| SA 总迭代 | $O(N_\text{base} + N_\text{percell} \cdot \|V_{\text{free}}\|)$ |
| 总计（含重启） | $O(R \cdot N \cdot WH)$ |

其中 $R$ 为重启次数，$N$ 为每次重启的 SA 迭代次数。

### 7. 参数说明

| 参数 | 默认值 | 含义 |
|------|--------|------|
| `iters_base` | 200,000 | SA 迭代次数的固定部分 |
| `iters_per_cell` | 25,000 | 每个可行走格子额外增加的迭代次数 |
| `restarts` | 8 | 独立重启次数 |
| `t0` | 3.0 | 初始温度 |
| `t_min` | 0.01 | 终止温度 |
| `seed` | 2024 | 随机种子（保证确定性输出） |

### 8. 输出

求解器输出包含：
- 最优覆盖度得分及对应的顾客路径
- ASCII 地图渲染（标注路径、货架、可见面数/朝向）
- SVG 可视化图（含正交视图和等距视图）
